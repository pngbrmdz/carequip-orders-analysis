# Care Quip Shopify Orders Analysis

This repo holds the Shopify orders export for Care Quip.

File: orders_export_1.xlsx (over 12,000 rows from 2017 to 2026)

## How to use
- Open orders_export_1.xlsx in Excel
- Or use the Python script to load it

## Files
- orders_export_1.xlsx → main data file
- requirements.txt → Python packages needed
- scripts/load_data.py → simple loader script
